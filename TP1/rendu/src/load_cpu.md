# load\_cpu(1) -- Load the CPU

## SYNOPSIS

`load_cpu` [`-s`] &lt;percentage&gt;

## DESCRIPTION

Load the CPU at &lt;percentage&gt;%.

## OPTIONS

  * `-s`:
    Silent the output.


## EXAMPLES

  * Load at 45%::
    `load_cpu 45`
